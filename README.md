# Zech Soft - Website

A premium corporate website built with modern web technologies.

## Technologies Used

-   **Frontend**: React + TypeScript
-   **Styling**: Tailwind CSS + shadcn/ui
-   **Animations**: Framer Motion
-   **Icons**: Lucide React
-   **Build Tool**: Vite

## Getting Started

### Prerequisites

-   Node.js (v18+)
-   npm

### Installation

```sh
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

-   **Premium UI**: Custom-designed sections with high-quality aesthetics.
-   **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
-   **Dark Mode**: Native dark mode support with a persistent toggle.
-   **Glassmorphism**: Modern glass effects throughout the site.
-   **Animations**: Smooth transitions and interactive elements using Framer Motion.
